# Motor Ativo - Plataforma de Vendas

Plataforma web para gerenciamento de campanhas de vendas automáticas.

## Funcionalidades

- 📞 Discador URA com IA
- 💬 Disparo WhatsApp (em breve)
- 📧 Disparo Email (em breve)
- 📊 Relatórios em tempo real
- 📁 Upload de planilhas

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000
